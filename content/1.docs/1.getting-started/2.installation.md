Get started with **Ungated**. Choose your platform below to find the specific installation steps.
:::tabs
  ::::tabs-item{label="Windows" icon="i-simple-icons-windows"}

  :::steps{level="4"}
  #### Download the Installer
  [`Ungated.exe`](https://github.com/gadanisor/ungated-releases/releases/latest/download/Ungated.exe)

  #### Verify download integrity (optional)
  To ensure your download hasn't been tampered with, you can verify the SHA256 checksum of the file.
  ```bash [powershell]
  Get-FileHash .\Ungated.exe -Algorithm SHA256
  ```
  Compare the output of the command with the value below
  ``` bash
  d2624846880c289fb92aed48207a6e7a3cf966e73c58caca570d2709cdf4574f
  ```
  #### Run the Executable
  Double-click the downloaded file. If Windows SmartScreen appears, click **"More info"** and then **"Run anyway"**.

  #### Complete Setup
  The application will install and launch automatically. A shortcut will be added to your Desktop and Start Menu.
  :::
  ::::

  ::::tabs-item{label="macOS" icon="i-simple-icons-apple"}
  :::steps{level="4"}
  #### Download the Disk Image
  [`Ungated-arm64.zip`](https://github.com/gadanisor/ungated-releases/releases/latest/download/Ungated-arm64.zip) [`Ungated-x64.zip`](https://github.com/gadanisor/ungated-releases/releases/latest/download/Ungated-x64.zip)

  Ungated runs natively on both Intel and Apple Silicon (M1/M2/M3) chips. Please select `Ungated-arm64.dmg` for Apple Silicon or `Ungated-x64.dmg` for Intel Macs.

  #### Verify download integrity (optional)
  To ensure your download hasn't been tampered with, you can verify the SHA256 checksum of the file.
  ```bash [terminal]
  shasum -a 256 Ungated-arm64.dmg
  # or
  shasum -a 256 Ungated-x64.dmg
  ```
  Compare the output of the command with the value below
  ``` bash
  arm64 de62f711a71bc83357ca9daa84ed40968ad77b11a8e5f2a60ab1f0c406d1ea0b
  x64 587f3783d90297d40b06024fe73f6db269a8877d98041af90ca2a27c274c1bfe
  ```
  #### Install to Applications
  Open the `.zip` file and drag the **Ungated** icon into your Applications folder.

  #### Grant Permission
  If you see a "Developer cannot be verified" message, go to `System Settings` and click **"Open Anyway"** under `Privacy & Security`.
  :::

  ::::tabs-item{label="Linux" icon="i-simple-icons-linux"}
  ### Install using the package manager (Recommended)

  Ungated provides an official APT repository for Debian-based distributions (Ubuntu, Linux Mint, Pop!_OS, etc).
  ::::steps{level="4"}
  #### Add the Ungated signing key
  ```bash
  curl -fsSL https://repo.ungated.net/ungated-archive-keyring.gpg \
  | sudo gpg --dearmor -o /usr/share/keyrings/ungated.gpg
  ```
  #### Add the repository for the beta distribution
  ```bash
  echo "deb [arch=amd64 signed-by=/usr/share/keyrings/ungated.gpg] https://repo.ungated.net beta master" \
  | sudo tee /etc/apt/sources.list.d/ungated.list
  ```
  #### Update package list
  ```bash
  sudo apt update
  ```
  #### Install Ungated
  ```bash
  sudo apt install ungated
  ```
  ::::
  ### Manual Download
  If you are using a non-debian linux distribution or just prefer a manual file download, please follow the steps below.
  :::steps{level="4"}
  #### Download the application
  [`Ungated.deb`](https://github.com/gadanisor/ungated-releases/releases/latest/download/Ungated.deb) [`Ungated.rpm`](https://github.com/gadanisor/ungated-releases/releases/latest/download/Ungated.rpm) [`Ungated.AppImage`](https://github.com/gadanisor/ungated-releases/releases/latest/download/Ungated.AppImage)

  ::note
  If you are not sure which one to pick, please download the `AppImage` version.
  ::

  #### Verify download integrity (optional)
  To ensure your download hasn't been tampered with, you can verify the SHA256 checksum of the file.
  ::code-group
  ``` bash [deb]
  sha256sum Ungated.deb
  ```
  ``` bash [rpm]
  sha256sum Ungated.rpm
  ```
  ``` bash [appimage]
  sha256sum Ungated.AppImage
  ```
  ::
  Compare the output of the command with the value below
  | Filename | SHA256 Checksum |
  | :--- | :--- |
  | Ungated.AppImage | `3cf49e24f956bbe6d7af1ff4dacade01cfadca20ea8efa181fcfa43e824bd8c8` |
  | Ungated.deb | `15600f304d377a8317353f6dbde6cd43fcb112d1cf75128a603bf3fc68aaa3ed` |
  | Ungated.rpm | `ea8cd69a47da14ad5de473cb422e7ce9621e8e53e478b9f8467f1444a0bc223d` |
  #### Install the app
    :::code-group
  
      ```bash [deb]
      sudo apt update
      sudo apt install ./Ungated.deb
      ```

      ```bash [rpm]
      # Install the .rpm package
      ```

      ```bash [appimage]
      # Make the file executable
      chmod +x Ungated.AppImage

      # Launch Ungated
      ./Ungated.AppImage
      ```
    :::

## Troubleshooting

:::accordion

::::accordion-item{label="macOS says “Ungated is damaged”" icon="i-lucide-circle-help"}
This can happen when Gatekeeper quarantines the app. Open System Settings (System Preferences), Privacy & Security, scroll down until you see a message about the blocked app and click Open Anyway.
::::

::::accordion-item{label="Linux AppImage doesn’t launch" icon="i-lucide-circle-help"}
Make sure the file is executable, then run it:

```bash
chmod +x Ungated.AppImage
./Ungated.AppImage
```
::::

::::accordion-item{label="Where are my projects stored?" icon="i-lucide-circle-help"}
Ungated projects live **where you create them** on your disk. Ungated works offline and doesn’t move your files unless you choose to.
::::

## System Requirements

- **macOS** 12+
- **Windows** 10+
- **Linux**: modern distro (AppImage recommended)
- **RAM**: 4 GB recommended
- **Disk**: ~300 MB free space

## Install

:::tabs{.w-full}

::::tabs-item{label="macOS" icon="i-lucide-apple"}

:::steps{level="4"}

#### Download the `.dmg`
Grab the macOS installer from the Download section.

#### Install the app
Open the `.dmg` and drag **Ungated** into **Applications**.

#### Open Ungated
Launch it from Applications.

#### If macOS blocks the app
Right-click **Ungated** → **Open** → confirm once.

:::

::::

::::tabs-item{label="Windows" icon="i-lucide-monitor"}

:::steps{level="4"}

#### Download the `.exe`
Grab the Windows installer from the Download section.

#### Run the installer
Double-click the `.exe` and follow the prompts.

#### Launch Ungated
Open Ungated from the Start Menu.

#### If Windows SmartScreen appears
Click **More info** → **Run anyway**.

:::

::::

::::tabs-item{label="Linux" icon="i-lucide-terminal"}

:::steps{level="4"}

#### Download the AppImage
Get `Ungated.AppImage` from the Download section.

#### Make it executable
```bash
chmod +x Ungated.AppImage
```

#### Run it
```bash
./Ungated.AppImage
```

#### Optional: move it somewhere permanent
```bash
mkdir -p ~/Applications
mv Ungated.AppImage ~/Applications/
```

:::

## First Launch

When Ungated opens for the first time, you can:

- **Create a new project**, or
- **Open an existing folder** (Markdown / plain text)

::tip
Don’t worry about setting up everything upfront. Start writing first — structure can grow alongside your project.
::

## Updates

Ungated updates won’t touch your projects.

- Download the latest release from the website, or
- Use in-app updates (when available)

## Troubleshooting

:::accordion

::::accordion-item{label="macOS says “Ungated is damaged”" icon="i-lucide-circle-help"}
This can happen when Gatekeeper quarantines the app. Remove quarantine attributes:

```bash
xattr -cr /Applications/Ungated.app
```
::::

::::accordion-item{label="Linux AppImage doesn’t launch" icon="i-lucide-circle-help"}
Make sure the file is executable, then run it:

```bash
chmod +x Ungated.AppImage
./Ungated.AppImage
```
::::

::::accordion-item{label="Where are my projects stored?" icon="i-lucide-circle-help"}
Ungated projects live **where you create them** on your disk. Ungated works offline and doesn’t move your files unless you choose to.
::::

:::

Continue to **Usage** to create your first project and write your first scene.

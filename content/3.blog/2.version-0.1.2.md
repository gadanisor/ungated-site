## A new step forward for Ungated

This update introduces several improvements aimed at expanding how writers manage projects and improving the editing experience inside Ungated.

The two main areas of focus are **project sharing** and **writing assistance**.

---

## GitHub project publishing

Projects can now be published directly to **GitHub** from inside Ungated.

This makes it possible to:

- access your projects from multiple machines
- collaborate with others
- use GitHub as a free remote backup for your writing

Once a project repository is initialized and at least one commit exists, Ungated will offer a **Publish on GitHub** option inside the Source Control panel.

After publishing, your project becomes a standard Git repository that can be cloned anywhere.

---

## Improved writing assistance

Ungated includes a built-in writing assistant powered by **Textlint**, and this update expands how it integrates with the editor.

### Configurable lint rules

Writers can now configure Textlint directly from the application settings.

Individual rules can be enabled or disabled depending on personal preference or project style.

Some of the available rules include:

- write-good (style suggestions)
- rousseau (typography and punctuation)
- unmatched punctuation detection
- duplicated conjunction detection

This allows writers to tailor the assistant to match their writing workflow rather than enforcing a rigid style.

---

## Editor quick fixes

Textlint suggestions can now be applied directly from the editor using **Quick Fix actions**.

When the writing assistant detects an issue and suggests a correction, the editor will display a quick action allowing the fix to be applied instantly.

This reduces friction when correcting small issues and helps maintain writing flow.

---

## New entity type: Item

Worldbuilding in Ungated now supports a new entity type: **Item**.

Items allow writers to track important objects within their stories, such as:

- artifacts
- weapons
- magical items
- tools
- story-critical objects

This expands the relationship model between story elements and makes it easier to structure complex fictional worlds.
